# 💱 Conversor Global de Moedas

Aplicação web para conversão de moedas em tempo real utilizando uma API pública de câmbio.

Projeto desenvolvido com foco em front-end puro, sem frameworks JavaScript, priorizando performance, visual moderno e código limpo.

---

## 🚀 Funcionalidades

- Conversão de valores entre diversas moedas internacionais  
- Taxas de câmbio atualizadas em tempo real  
- Seleção de moeda de origem e destino  
- Botão para troca rápida de moedas  
- Exibição da cotação atual  
- Exibição da última atualização  
- Interface responsiva (desktop e mobile)  
- Animações e feedback visual  
- Tratamento de erros de conexão
